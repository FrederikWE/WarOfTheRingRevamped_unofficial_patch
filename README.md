This patch is supposed to make War of the Ring REVAMPED work with version 9.3 of Age of the Ring mod for Battle for Middle-Earth 2: Rise of the Witch King.

I take no credibility in creating this mod, only in "fixing" it for the latest version of Age of the Ring (9.3).

The original creator is TheLordofAngmar and the official page for this mod can be found on: https://www.moddb.com/mods/war-of-the-ring-revamped-an-aotr-submod .

This patch is not stable and might still cause crashing, I will attempt to continue improving untill the original creator returns.

The uploaded version is the most stable. I am unsure whether or not the mod will work if you run / apply any of the settings..

Normal patched version is version 3.6.0 . 

I did add one version that's slightly tweaked (version 3.6.1). Included in this version:

- Added costs to buildings on the strategic map
- Increased money gain from buildings on the strategic map

To install:

1. Create a new mod in the AOTR Launcher

2. Open the mod folder

3. Upload everything from the main folder (without .git, .gitignore and other git files) to the modfolder

4. Enable the mod

5. Launch AOTR

Known issues (to be fixed by v3.6.0.1 and v3.6.1.1):
